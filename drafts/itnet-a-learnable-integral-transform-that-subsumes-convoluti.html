Key Details

  • The ITNet layer computes output y_i = ∫ K(x_i, x_j; θ) · f_j dx_j, where K is an MLP taking concatenated position and feature vectors.
  • When K depends only on relative position and is shift‑invariant, the layer reduces to a standard convolution.
  • When K is a function of query‑key dot product (content) and a fixed positional encoding, the layer reproduces multi‑head self‑attention.
  • When K is triangular (causal) and depends only on past positions, the layer mimics a recurrent update.
  • The MLP kernel is typically two hidden layers with 64 units each, keeping overhead low compared to full attention matrices.
  • Experiments on image classification (CIFAR‑10), language modeling (WT‑2), and speech recognition show ITNet matches or slightly exceeds the baseline specialized models while using a single architecture.
